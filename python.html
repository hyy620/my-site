Python基础课程

从零开始学习Python编程

课程介绍

本课程旨在帮助初学者快速掌握Python编程基础，包括语法、数据类型、控制结构、函数、面向对象编程等核心内容。通过理论学习和实践练习，你将能够编写简单的Python程序，为进一步学习打下坚实的基础。

课程内容

第一章：Python入门

  • Python简介
  • 安装Python
  • 第一个Python程序
  • IDE选择
# 第一个Python程序 print("Hello, Python!")

第二章：数据类型

  • 数值类型
  • 字符串
  • 列表
  • 元组
  • 字典
# 基本数据类型 a = 10 # 整数 b = 3.14 # 浮点数 c = "Python" # 字符串 d = [1, 2, 3] # 列表 e = (4, 5, 6) # 元组 f = {"name": "Python"} # 字典

第三章：控制结构

  • 条件语句
  • 循环语句
  • break和continue
# 条件语句 if x > 0: print("正数") elif x < 0: print("负数") else: print("零") # 循环语句 for i in range(5): print(i)

第四章：函数

  • 函数定义
  • 参数传递
  • 返回值
  • lambda函数
# 函数定义 def greet(name): return f"Hello, {name}!" # 调用函数 print(greet("Python"))

第五章：面向对象编程

  • 类和对象
  • 继承
  • 多态
  • 封装
# 类定义 class Person: def __init__(self, name): self.name = name def greet(self): return f"Hello, I'm {self.name}" # 创建对象 p = Person("Alice") print(p.greet())

第六章：模块和包

  • 导入模块
  • 创建模块
  • 包的结构
# 导入模块 import math print(math.pi) # 从模块导入特定函数 from random import randint print(randint(1, 10))
