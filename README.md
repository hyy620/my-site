## 同步到GitHub

### 方法一：使用git命令（推荐）

1. 安装git
2. 打开命令行，进入项目目录
3. 执行以下命令：

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/hyy620/my-site.git
git push -u origin main
```

### 方法二：手动上传

1. 访问 https://github.com/hyy620/my-site.git
2. 点击 "Add file" -> "Upload files"
3. 上传 `index.html` 文件
4. 提交更改
